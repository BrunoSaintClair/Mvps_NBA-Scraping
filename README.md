# Mvps_NBA-Scraping

## Esse mini-projeto, feito com intuito de revisar algumas coisas, foi dividido nas etapas de: 
* acesso ao site oficial da NBA
* raspagem e tratamento de dados
* apresentação gráfica 

## Foram utilizadas as bibliotecas: 
* Requests
* BeautifulSoup
* Pandas
* MatPlotLib

Além de RegEx.
